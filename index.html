Pomodoro

To use the pomodoro timer write the time in minutes and press the start button. You can stop the timer at any time by pressing the stop button. Also, you can activate/desactivate the finish notification with the sound button.

To do List

To start using the To Do List you can add a new task by writting it and pressing the add task button. You can delete the task by clicking on the remove button. Also, you can move the task to the other task status by dragging it.
